# CEIA_PLN_Desafios

Repositorio de desafíos de Procesamiento de Lenguaje Natural para CEIA.

## Integrante

- Rivollier Gaspar

## Descripción

Este repositorio reúne distintos desafíos de PLN. Por ahora solo está disponible el primer desafío, que se encuentra en el notebook correspondiente.

## Estructura

- [Notebooks/Desafio_1.ipynb](./Notebooks/Desafio_1.ipynb): Primer desafío de PLN.
- [Notebooks/Desafio2.ipynb](./Notebooks/Desafio_2.ipynb): Segundo desafío de PLN.

## Uso

1. Instalar dependencias (si es necesario) con `uv`:
   - `uv add numpy pandas matplotlib scikit-learn nltk` 
2. Abrir el notebook `Notebooks/Desafio_1.ipynb`
3. Ejecutar las celdas en el orden especificado

> Si ya están instaladas, omitir el paso 1.
